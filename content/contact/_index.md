Nom Zine is maintained as an independent learning and resource project. For corrections, resource suggestions, or collaboration notes, contact the editor through the community links in the footer.

If you are reporting an issue with a source, font, image, or Chữ Nôm reading, include the page URL and a short explanation so it can be checked carefully.
